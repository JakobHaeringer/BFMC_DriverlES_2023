Detailed Description

This class implements the translation from the Raspberry Pi ROS messages to messages that the STM board can handle.

Since the STM board isn't running ROS, the data that are transmitted to the STM board first need to be mapped from the ROS message type to a custom message type. For this, the ROS topic command is being subscribed. By distinguishing the command the handler then triggers the function that sends the command to the STM board.
