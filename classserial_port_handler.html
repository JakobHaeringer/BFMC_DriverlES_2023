Detailed Description

This class handles the sending of the messages to the STM board via serial and it forwards the received messages to the ResponseHandler.

The lower level functionality that is responsible for handling the messages is implemented in this serialPortHandler. It is responsible for reading/writing messages from/to the buffer. It also establishes/closes the connection with the STM board. The serialPortHandler is used by the higher level class serialCommunicationHandler.
