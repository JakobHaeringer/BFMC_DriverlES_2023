Detailed Description

This class holds the functions that sends commands to the STM board.

By opening a connection to the STM board via serial, the Raspberry Pi is able to send commands to the STM board, which will the execute them and apply it to the actuators on the vehicle. For this custom messages are designed, that contain a key and a body. The key identifies what command to execute and the body holds the information on how to control the actuator.
