Detailed Description

The header file for the output commands used in the BFMC 2023.

Author
Jakob Haeringer

Config file for steering and dc motor constraints on the Nucleo Board in BFMC/Embedded_Platform/src/main.cpp.
Steering constraints -22° bis 22° because of the chassis - -23 to +23 on Nucleo Board.
Speed constraints - Signal for DC Motor -0.3 to 0.3 (m/s) on Nucleo Board.
